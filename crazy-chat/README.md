# CrazyChat web

## 依赖
  ### vant ui : https://vant-contrib.gitee.io/vant/v2/#/zh-CN/toast

## 若文件缺少node_modules 
```shell
cd web_crazychat
npm install
```

## 开发服务启动
```shell
npm run serve
```

## 上线部署 (暂时用不到)
```
npm run biuld
```
## 文件夹

- src
  + api 书写api接口的模块 axios
  + assets 静态资源文件
  + router vue-router 路由模块
  + store vueX 数据管理模块
  + views 路由组件模块
  + components 全局组件模块
  + APP.vue 
  + main.js 入口文件
